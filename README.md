# Flo — Expense Tracker

A professional full-stack expense tracker built with Flask + vanilla JS.

## Setup & Run

```bashe
# Install dependencies
pip install flask werkzeug

# Run the app
python app.py
```

Then open http://localhost:5000 in your browser.

## Features
- 🔐 User authentication (signup / login)
- 📊 Beautiful dashboard with real-time charts
- 💸 Add / edit / delete expenses
- 🗂️ 7 categories: Food, Transport, Shopping, Entertainment, Health, Utilities, Travel
- 📈 Analytics page with 4 chart types
- 🌙 Dark theme with professional UI

## Tech Stack
- **Backend**: Python Flask, SQLite
- **Frontend**: Vanilla HTML/CSS/JS, Chart.js
- **Auth**: Werkzeug password hashing + Flask sessions
